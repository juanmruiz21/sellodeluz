# 📸 Sello de Luz | Por Juano Ruiz

**Sello de Luz** es una herramienta web diseñada para fotógrafos que buscan proteger y profesionalizar sus capturas de forma rápida. Esta aplicación extrae automáticamente los metadatos **EXIF** de tus fotografías y genera una firma estética con los parámetros técnicos de la toma.

## 🚀 Características
- **Extracción EXIF:** Visualiza automáticamente Apertura (ƒ), Velocidad, ISO y Distancia Focal.
- **Marca de Agua Profesional:** Diseño minimalista con tipografía de alta legibilidad.
- **Código QR Dinámico:** Vincula cada foto directamente a tu portfolio de Instagram.
- **Privacidad Total:** El procesamiento se realiza localmente en el navegador; tus fotos nunca se suben a un servidor.

## 🛠️ Tecnologías Usadas
- **HTML5 / CSS3 / JavaScript** (Vanilla)
- **Exifr.js:** Para el parseo de metadatos de imagen.
- **QRCode.js:** Para la generación del código QR.
- **GitHub Pages:** Para el despliegue automático.

## 📖 Cómo usarlo
1. Sube tu fotografía en formato JPG/JPEG.
2. Haz clic en "Generar Sello Profesional".
3. Descarga o comparte tu obra con los datos técnicos integrados.
